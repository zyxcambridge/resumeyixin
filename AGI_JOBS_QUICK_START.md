## 📝 编译PDF简历

在您的本地机器上运行以下命令编译PDF：

```bash
# 安装LaTeX (Ubuntu/Debian)
sudo apt-get install texlive-xetex texlive-fonts-recommended

# 安装LaTeX (macOS)
brew install --cask mactex

# 编译所有简历为PDF
for f in resume_agi_*.tex; do
    xelatex "$f"
done
```
